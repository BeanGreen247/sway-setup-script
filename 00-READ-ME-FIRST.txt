GETTING STARTED
═══════════════════════════════════════════════════════════════════════════

OPTION A: Transfer files to your Debian system
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  On Windows (copy files to USB):
    > Copy-Item C:\tmp\* -Destination D:\sway-install\

  On Debian (from USB):
    $ mkdir -p ~/sway-install
    $ cp /media/usb/sway-install/* ~/sway-install/
    $ cd ~/sway-install
    $ chmod +x *.sh
    $ sudo bash master-install.sh

OPTION B: Use the files as reference
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Copy the script contents to your GitHub repo and run from there:
    $ git clone https://github.com/BeanGreen247/sway-setup-script.git
    $ cd sway-setup-script
    $ chmod +x *.sh
    $ sudo bash master-install.sh

═══════════════════════════════════════════════════════════════════════════
INTEL HD 530 OPTIMIZATIONS
═══════════════════════════════════════════════════════════════════════════

  Kernel Parameters (/etc/modprobe.d/i915.conf):
     enable_rc6=1 CPU deep sleep states
     enable_fbc=1 Framebuffer compression
     enable_psr=1 Panel self-refresh (power save)
     fastboot=1 Skip mode setting on boot
     enable_guc=2 GuC firmware loading
     enable_dc=2 Display C-states

  Sway Performance Settings:
     focus_follows_mouse no Reduce CPU wake-ups
     mouse_warping none Prevent cursor updates
     max_render_time 3 Limit frame time for iGPU

  VAAPI Hardware Acceleration:
     LIBVA_DRIVER_NAME=i965
     Hardware video decode/encode
     Reduced CPU usage for video

    Super + N Toggle notification center

VERIFICATION COMMANDS (After Install)
═══════════════════════════════════════════════════════════════════════════

  Run the verification script:
    $ bash verify-installation.sh

  Manual checks:
    $ free -h Check RAM usage (~200MB)
    $ htop Monitor CPU usage (<5% idle)
    $ sensors Check temperatures
    $ glxinfo | grep renderer Verify Mesa driver
    $ wpctl status Check audio devices
    $ swaymsg -t get_outputs List displays

═══════════════════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

  SWAY WON'T START:
    $ journalctl --user -xe | grep sway
    $ sway --validate
    $ sway --debug 2>&1 | tee ~/sway-debug.log

  AUDIO NOT WORKING:
    $ systemctl --user restart pipewire pipewire-pulse wireplumber
    $ wpctl status
    $ pavucontrol

  SCREEN TEARING:
    $ cat /sys/module/i915/parameters/enable_fbc # Should be 1
    $ cat /sys/module/i915/parameters/enable_psr # Should be 1
    $ sudo update-initramfs -u -k all && sudo reboot

  FONTS MISSING:
    $ fc-cache -f -v
    $ sudo apt install --reinstall fonts-font-awesome
